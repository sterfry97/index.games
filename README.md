## 📖 What is Index?

**Index** is a fast, clean, and fully unblocked website packed with everything you need — web proxy, search engines, games, tab cloaking, and more. Whether you're trying to get around network restrictions or just want a smooth browsing experience, Index has you covered.

---

## ✨ Features

- 🔍 **Search Engines** — Search directly with Google, DuckDuckGo, and more without restrictions
- 🌐 **Web Proxy** — Browse sites like TikTok, YouTube, and the rest of the web freely
- 🎮 **Games** — A built-in collection of unblocked games to play anytime
- 🕵️ **Tab Cloaking** — Disguise your tab title and favicon to stay under the radar
- 🚀 **Unblocked** — Bypasses common network filters and firewalls
- ⚡ **Fast & Clean UI** — Minimal, responsive design that just works

## 🚀 Getting Started

### Visit the Site
Simply navigate to the hosted URL and start browsing — no installs required.

### Self-Hosting
```bash
# Clone the repository
git clone https://github.com/sterfry97/index.games.git
```

> For full proxy functionality, deploy to a platform like **Vercel**, **Netlify**, or **Cloudflare Pages**.
